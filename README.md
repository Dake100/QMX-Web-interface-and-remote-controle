This project is based on the work of Randy Rollins, KD7RR, published on https://github.com/rrollins72/QMX-Serial-remote 
In that project Randy uded a ESP32-S3 Zero, a small TFT display, a rotary knob and a cople of buttons to remotly watch and controle some of the functions om the ham radio transceiver QMX+.

In this new project, by Dan Åkerlund SM0EHV and Claude.AI, a WiFi-based web interface has been added, allowing the transceiver to be monitored and controlled from any web browser on the local network — on a desktop or laptop computer, tablet, or smartphone. 
This new web interface mirrors the information that is shown on the TFT display in the previous project, plus a litle bit more, togeter with full remote control of all key functions on QMX+. With the mouse weel, you can adjust the frequensy and scroll trough the band.
